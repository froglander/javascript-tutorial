How to tackle a programming assignment

Let's build Hangman

How do you play Hangman?

Imagine playing Hangman with your friends when you were in grade school, how did it work?

Well, if I am the one playing Hangman, I see a blanked out word(s) that has placeholders showing how many letters are in the word(s)

I guess a letter, if it's in the word, it is written in the correct location on the placeholder line, if it is not in the word a body part gets added to the gallows until I either guess the word or run out of body parts

What if you are the one who sets the game up?

Well, if I am the one who is setting the game up, I think of a word, draw underscores on the paper to show how many letters there are and then write down when the person playing gets a letter right or wrong

Pseudocode!

The computer is going to take the part of the person who sets the game up, let's walk through the process

  • First, we need a list of words
  • Next, we need to be able to pick a random word
  • Then, we need to capture the letter the user presses
  • Next, we check if the letter is in the word
    • If it is, we show it on the screen
    • If it is NOT, we add a body part to the gallows (or decrement the remaining tries)
  • Rinse and repeat!
