Basic Coding Vocabulary

Term What is it? Example
Variable A way to store a value var name = "Bob";

This means we have created a container to store a value and we have assigned the container "name" the value "Bob"

Array An array is a way to store a list of values. You could have an array of strings, an array of numbers, an array of arrays, etc. var myArray = [];

In this example, we have declared an empty array called myArray. At this point, space has been set aside in memory for it, but it contains no values.

var myOtherArray = ["cat", "dog", "horse"];

In this example, we have declared an array called myOtherArray and assigned it some string values.
